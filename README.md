# PolyvLiveAPI

当前版本：

- PolyvLiveAPI ~ [0.1.3](http://repo.polyv.net/ios/download/liveAPI/0.1.3/PLVLiveAPI.framework.zip)

Podfile 中添加

```ruby
platform :ios, "8.0"

use_frameworks!

target 'YourApp' do
    
	pod 'PolyvLiveAPI', '~> 0.1.3'      # Polyv live api.
end
```
